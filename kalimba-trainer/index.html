アプリ概要

Kalimba Trainer は、17キーのカリンバ練習に特化したトレーニングアプリです。 ノーツ落下表示に合わせた演奏練習、MIDI取り込み再生、再生速度調整、キー位置調整に対応しています。

主な機能

  • 17キー配列の演奏表示
  • 内蔵曲の再生練習
  • MIDIファイルの取り込みと再生
  • 再生速度の調整
  • キー位置調整

Overview

Kalimba Trainer is a practice app focused on 17-key kalimba. It supports falling-note practice, MIDI import/playback, playback speed control, and key position calibration.

Main Features

  • 17-key performance display
  • Built-in song practice
  • MIDI import and playback
  • Playback speed control
  • Key position calibration
